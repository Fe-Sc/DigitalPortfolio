Creature project

This project was my first introduction to programming, where i learnt the basics of Java Processing.

The project was made for the Introduction to Programming course in the first module of the first year of my Creative Technology study.

We were tasked to make a "creature" that could interact with the user, making use of functions, arrays and classes.

I chose to create a bird creature that is able to fly and move around the canvas.
